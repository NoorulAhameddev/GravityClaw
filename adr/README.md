# Architecture Decision Records (ADR)

This directory contains Architecture Decision Records (ADRs) for Gravity Claw. ADRs document significant architectural decisions, their context, and consequences.

## What is an ADR?

An ADR is a short document capturing:

- **Context** — What's the problem or motivation?
- **Decision** — What did we decide?
- **Consequences** — What are the trade-offs and impacts?
