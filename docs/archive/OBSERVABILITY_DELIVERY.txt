## 📌 ENVIRONMENT VARIABLES

Add to `.env`:

```bash
# Logging
LOG_LEVEL=info
LOG_FORMAT=pretty
ENABLE_CALLER_INFO=false

# Metrics
ENABLE_METRICS=true
ENABLE_METRICS_PERSISTENCE=false
METRICS_RETENTION_HOURS=24

# Correlation & Tracing
CORRELATION_ID_HEADER=X-Correlation-ID
ENABLE_TRACING=true

# OpenTelemetry (optional)
OTEL_ENABLED=false
# OTEL_EXPORTER_OTLP_ENDPOINT=http://localhost:4318
```

### Prometheus Metrics
```bash
GET /metrics
```
**Response:** Prometheus text format
**Use with:** Prometheus scraping

### JSON Metrics
```bash
GET /api/metrics
```
**Response:** Structured JSON metrics
**Use with:** Dashboards, APIs

### Trace Details
```bash
GET /api/traces/{correlationId}
```

**Use with:** Debugging
