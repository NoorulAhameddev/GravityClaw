# GravityClaw Node.js Client SDK

The `@gravityclaw/client` SDK provides a robust, typed interface for interacting with your GravityClaw agent cluster. It abstracts away the complexities of WebSocket management, HTTP fallback, and response streaming.

## Installation

You can install the SDK via npm, yarn, or pnpm.

```bash
npm install @gravityclaw/client
```

## Quick Start

The quickest way to get started is by initializing the `GravityClawClient` and sending a message.

```typescript
import { GravityClawClient } from '@gravityclaw/client';

async function main() {
  const client = new GravityClawClient({
    apiKey: process.env.GRAVITYCLAW_API_KEY, // Or tenant token
    baseUrl: 'https://api.your-gravityclaw-instance.com',
  });

  const response = await client.chat.sendMessage({
    message: 'Hello, world!',
    sessionId: 'user-session-123'
  });

  console.log(response.text);
}

main();
```

## Streaming Responses

For a more responsive user experience, you can stream tokens as they are generated by the LLM.

```typescript
import { GravityClawClient } from '@gravityclaw/client';

async function streamChat() {
  const client = new GravityClawClient({
    apiKey: process.env.GRAVITYCLAW_API_KEY,
    baseUrl: 'https://api.your-gravityclaw-instance.com',
  });

  const stream = await client.chat.streamMessage({
    message: 'Write a poem about the sea.',
    sessionId: 'user-session-456'
  });

  for await (const chunk of stream) {
    if (chunk.type === 'token') {
      process.stdout.write(chunk.content);
    } else if (chunk.type === 'tool_call') {
      console.log(`\n[Agent invoked tool: ${chunk.toolName}]`);
    }
  }
}

streamChat();
```

## Error Handling

The SDK provides typed errors that you can catch and handle gracefully.

```typescript
import { GravityClawClient, APIError, AuthenticationError } from '@gravityclaw/client';

try {
  await client.chat.sendMessage({ message: 'Hello' });
} catch (error) {
  if (error instanceof AuthenticationError) {
    console.error('Invalid API Key');
  } else if (error instanceof APIError) {
    console.error(`API Error: ${error.status} - ${error.message}`);
  } else {
    console.error('Unknown error:', error);
  }
}
```

## Best Practices

1. **Keep `sessionId` consistent**: To maintain conversational context and memory across multiple turns, use the same `sessionId` string for a given user or conversational thread.
2. **Environment Variables**: Never hardcode your API key. Always use environment variables (`process.env.GRAVITYCLAW_API_KEY`).
3. **Handle Disconnects**: If using the WebSocket underlying streaming transport, make sure to implement retry logic (the SDK handles basic exponential backoff by default).
